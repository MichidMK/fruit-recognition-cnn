# Fruit Recognition System using CNN

A deep learning-based fruit and vegetable recognition system built for my Bachelor's thesis at Anhui University of Technology.

## Overview
- Recognizes 10 classes of fruits and vegetables
- Baseline CNN vs MobileNetV2 transfer learning comparison
- 98%+ accuracy on test dataset
- Exported to TFLite for mobile deployment

## Technologies
- Python
- TensorFlow / Keras
- MobileNetV2 (Transfer Learning)
- OpenCV
- Google Colab

## Results
| Model | Parameters | Accuracy |
|-------|-----------|----------|
| Baseline CNN | 1.73M | 100% (test) |
| MobileNetV2 | 165K trainable | 95% (real-world) |

## Dataset
4,522 training images / 1,339 test images across 10 classes
